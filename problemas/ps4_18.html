PS 4.18 - Generar Matriz Identidad

Problema PS 4.18 (pág. 446)

Generar Matriz Identidad

Construya un diagrama de flujo que llene de ceros una matriz cuadrada A [N x N], excepto en la diagonal principal donde debe asignar 1.

Requisitos

  • Datos: N (dimensión de la matriz, $1 < N < 50$).
  • Crear una matriz A[N][N].
  • Usar ciclos anidados (para $i$ y $j$).
  • Si $i == j$ (diagonal principal), asignar 1.
  • Si $i != j$, asignar 0.
  • Imprimir la matriz resultante.

Notas rápidas

  • Este ejercicio es clave para entender el manejo de índices (i, j) en matrices.
  • La condición i == j es la definición de la diagonal principal.
  • No hay diagrama de flujo en el libro para este problema.
