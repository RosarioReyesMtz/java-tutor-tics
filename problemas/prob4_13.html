Problema 4.13 - Intercambiar Renglones de Matriz

Problema 4.13 (pág. 287)

Intercambiar Renglones de una Matriz Cuadrada

Construya un diagrama de flujo que intercambie los renglones de un arreglo bidimensional cuadrado (A[N x N]). El renglón 1 debe intercambiarse con el N, el 2 con el N-1, y así sucesivamente.

Requisitos

  • Datos: Matriz A[N, N] (tipo entero, $1 < N < 50$).
  • El intercambio es simétrico: Renglón $i$ se intercambia con $N - 1 - i$ (en base 0) o $N - i + 1$ (en base 1).
  • El ciclo principal debe iterar solo hasta la mitad de los renglones ($N / 2$).
  • Se debe usar una variable $AUX$ para el intercambio.

Notas rápidas

  • El bucle exterior (i) solo necesita llegar a N / 2. Si llega hasta N, volvería a intercambiar los renglones, revirtiendo la operación.
  • El bucle interior (j) debe recorrer todas las columnas.
